# Find Minimum Element and Its Index

## Problem Statement
You are given an integer array `arr` of size `length`.  

Implement a function to find the **minimum element** in the array and print:
1. The minimum element  
2. Its index  

Both should be printed on **separate lines**.

---

## Notes
- Array index starts from **0**
- There will be **only one minimum element**
- Do not print any extra text

---

## Input Format
- First line contains an integer `length`
- Second line contains `length` integers (array elements)

---

## Output Format
- First line → minimum element  
- Second line → index of the minimum element  

---

## Sample Input
```
10
23 45 82 27 66 12 78 13 71 86
```

## Sample Output
```
12
5
```

---

## Explanation
Array:
```
[23, 45, 82, 27, 66, 12, 78, 13, 71, 86]
```

Minimum element = **12**  
Index = **5**

---

## Time Complexity
O(n)

## Space Complexity
O(1)
