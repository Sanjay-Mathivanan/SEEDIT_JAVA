# Day 03 - Value of Cards

## 📅 Date
08 March 2026

## 🧩 Problem
You are given four cards from a deck represented by numbers.

Card Representation:

Ace → 1  
2 → 2  
3 → 3  
...  
10 → 10  
Jack → 11  
Queen → 12  
King → 13  

In the card game **Twenty-Nine**, points are assigned as follows:

Jack → 3 points  
9 → 2 points  
Ace → 1 point  
10 → 1 point  
All other cards → 0 points  

The task is to calculate the **total points of the given four cards**.

---

## 💡 Approach

1. Read four integers representing the cards.
2. Check the value of each card.
3. Assign points according to the rules.
4. Add all the points.
5. Print the final total.

---

## 📥 Input


11
9
1
5


---

## 📤 Output


6


---

## 📖 Explanation

11 → Jack → 3 points  
9 → 2 points  
1 → Ace → 1 point  
5 → 0 points  

Total = **3 + 2 + 1 + 0 = 6**
